# Diagramas

Esta carpeta contiene los diagramas del proyecto.

## Tipos de Diagramas

### 1. Diagrama de Arquitectura

- Muestra la estructura general del sistema
- Componentes principales y sus relaciones
- Flujo de datos entre módulos

### 2. Diagrama de Clases

- Clases principales del sistema
- Relaciones entre clases
- Métodos y atributos relevantes

### 3. Diagramas de Flujo

- Flujo de ejecución de operaciones principales
- Proceso de consulta a APIs
- Proceso de web scraping
- Proceso de lectura de RSS
- Flujo de la interfaz de usuario

### 4. Diagrama de Secuencia

- Interacciones entre componentes
- Llamadas a APIs externas
- Flujo de datos desde la fuente hasta el CSV

## Herramientas Recomendadas

- draw.io / diagrams.net (online, gratis)
- PlantUML (text-based)
- Lucidchart
- Microsoft Visio
- Google Drawings

## Formato

Guardar los diagramas en formatos estándar:

- PNG (para visualización)
- SVG (vectorial, escalable)
- Archivo fuente editable (.drawio, .puml, etc.)
