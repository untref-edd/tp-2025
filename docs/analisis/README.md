# Análisis y Diseño

Esta carpeta contiene el análisis y diseño de las técnicas implementadas en el proyecto.

## Contenido

### Análisis de Técnicas

Para cada técnica implementada, documentar:

1. **APIs (OpenAlex y The Lens)**
   - Análisis de la documentación de las APIs
   - Endpoints utilizados
   - Estructura de datos de respuestas
   - Estrategias de paginación
   - Manejo de rate limits
   - Estructuras de datos utilizadas para almacenar y procesar información

2. **Web Scraping**
   - Análisis de la estructura HTML de cada sitio
   - Selectores CSS/XPath utilizados
   - Estrategias para manejar contenido dinámico
   - Manejo de paginación
   - Estrategias de manejo de errores
   - Estructuras de datos para consolidar información

3. **RSS Feeds**
   - Análisis de la estructura XML de los feeds
   - Extracción de elementos específicos
   - Procesamiento de fechas y metadatos
   - Estrategias para identificar países asociados
   - Estructuras de datos para almacenar noticias

### Análisis de Complejidad

Documentar la complejidad algorítmica de las operaciones principales:

- Complejidad temporal
- Complejidad espacial
- Optimizaciones implementadas

### Comparación de Enfoques

Comparar las diferentes técnicas utilizadas:

- Ventajas y desventajas de cada método
- Casos de uso apropiados
- Lecciones aprendidas
